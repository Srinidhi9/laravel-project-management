# 🗂️ Project Management System
**Laravel | REST APIs | MySQL**

A web-based **Project & Task Management System** built using Laravel that allows users to manage projects, track tasks, and collaborate efficiently with role-based access.

This application demonstrates backend development skills including **REST API creation, authentication, database design, validation, and MVC architecture**.

---

## 🚀 Features

- User Authentication (Login & Registration)
- Role-based Access Control (Admin / User)
- Project Creation & Management
- Task Assignment & Status Tracking
- CRUD operations via REST APIs
- Form validation & input sanitization
- Secure database interactions using Eloquent ORM
- MVC architecture implementation

---

## 🛠️ Tech Stack

- **Backend:** PHP, Laravel  
- **API:** RESTful APIs  
- **Database:** MySQL  
- **Architecture:** MVC Pattern  
- **Tools:** Postman, Git, GitHub, VS Code  

---

## ⚙️ Setup Instructions

### 1.Clone the repository
git clone https://github.com/Srinidhi9/laravel-project-management.git

### 2.Install dependencies
composer install

### 3.Configure environment
cp .env.example .env
php artisan key:generate

### 4.Run migrations
php artisan migrate

### 5.Start server
php artisan serve
